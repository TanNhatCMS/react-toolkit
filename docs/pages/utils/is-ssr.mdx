`isSSR` is a utility to distingush between server environment and client
environment.

## Import

```js
import { isSSR } from '@dwarvesf/react-utils'
```

## Return value

This method returns `true` if the function is invoked in a browser and returns
`false` if it is invoked in a server.

## Usage

```js
if (isSSR()) {
  /// do something
}
```
