`useLockBodyScroll` is a custom hook that handles copying content to clipboard.

## Import

```js
import { useLockBodyScroll } from '@dwarvesf/react-hooks'
```

## Return value

The `useMergeRefs` hook returns a function that receives the element and assign
the value to the given React refs.

## Usage

```jsx
function Example({ ref, ...props }) {
  const internalRef = React.useRef()
  const refs = useMergeRefs(internalRef, ref)

  return (
    <div {...props} ref={refs}>
      A div with multiple refs.
    </div>
  )
}
```
